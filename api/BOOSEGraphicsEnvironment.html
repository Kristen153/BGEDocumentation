Namespace BOOSEGraphicsEnvironment

Classes

AppCanvas

Represents the drawing canvas for the BOOSE graphics environment. Provides drawing operations such as lines, shapes, text, and canvas management.

AppCommandFactory

Factory class responsible for creating Command instances based on command type strings. Validates the command type and throws a BOOSE.FactoryException for invalid commands.

AppForm

Main form for the BOOSE graphics environment application. Handles front facing user interactions, delegates command parsing and execution to the Factory, and updates the canvas and command log.

BitmapManager

Manages a bitmap and its associated graphics context. Provides functionality to create, copy, resize, and dispose of the bitmap.

Interfaces

IDrawable

Represents an object that can be drawn on a canvas.
